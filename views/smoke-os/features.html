SmokeOS Features

#{include '/smoke-os/mn.html' /}

Characteristics

The kernel is build in C and assembly and design as a monolithic kernel architecture.

SmokeOS system is licensed under Affero GPL.

The system is multitask and multithread.

The kernel have CPU detection and use virtual memory.

File Systems & Devices

Device detected: ATA, ATAPI. Support of MBR & GPT partitions.

File systems supported are: ISO, FAT and a basic TMPFS which is different from linux.

Other devices are Keyboard and a basic framebuffer.

Platform

This system is build only for i386 architecture yet.

I study how to run on RaspberryPi for my first ARM port.

There is a Usermode architecture supported that allow to simulate syscall to a kernel program. Those are used as functional tests (launch *.strace files).
