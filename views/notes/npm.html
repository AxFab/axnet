My Node.Js Packages

I developed several packages for nodejs that have been published on npm.

Axml     Github   Npm

Node.js package, Xml Parser - convert Xml to JsonML.

This small js library have been build to parse XML. Most of xml library I study in Js is loosing some precious informations (children order, blank text).

I decided to create my own XML parser which, by default provide JsonML object but can also be override to build a custom document

Srpc     Github   Npm

An easy to use RPC implementation using json-rpc specification.

SRPC allow to share method across network using HTTP request.

The configuration take only two method to make a server and a client to communicate. The client receive an object which share its methods with the one defined on the server.

TinyDB     Github   Npm

Small document database. Create a store of JSON documents.

Tiny DB is a JSON document store database. The API is based on MongoDB.

Html-Markup     Github   Npm

Node.js package, Html template engine.

This is an engine that define a new HTML template language. The language used is strongly base on the one used on the first iteration of the play framework.

This lanaguage is far from the best available but I designed it because I needed file inclusion.
